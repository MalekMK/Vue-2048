# Vue-2048

## Description

This application is a 2048 game where player merges tiles together to create tile 2048 to win.

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run serve
```

### Compiles and minifies for production
```
yarn run build
```

### Run your tests
```
yarn run test
```
